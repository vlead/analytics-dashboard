* Introduction
  - This document explains the realization plan for
    implementing =analytics-dashboard= using =feedback
    analytics= data.

* Plan ([[https://github.com/vlead/analytics-dashboard/milestone/2][Milestone])
  :PROPERTIES:
  :CUSTOM_ID: plan-for-milestone2
  :END: 
  Once this milestone is realized, the following are done to
  the Analytics Dashboard:
  1. dynamically displaying the total number of feedbacks on
     whole.
  2. dynamically display yearly wise graphs.
  3. dynamically visualize feedback responses of particular
     year w.r.t lab, exp and generic feedbacks.
  4. dynamically visualize feedback responses w.r.t OL, NOL,
     OE, NOE and generic feedbacks.
  5. Transition of graphs and arrange them properly in a
     single page.
